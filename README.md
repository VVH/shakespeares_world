# Transcribe

A web full-text transcription app written in [AngularJS](https://angularjs.org/).

## Requirements

* Node / NPM

## Installing locally

1. Clone the repo
2. Run `npm install`

## Running locally

1. Run `npm start` (an alias for `gulp`)
